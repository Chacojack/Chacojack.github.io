发表于 2016-07-07   |   分类于 Android   |     |   阅读次数

在应用发布的时候，可能面临多种渠道发布的问题，为了对不同渠道做统计，需要在应用中表明渠道，要是一个一个去改效率很低。Android Studio为我们提供了简便的方法。
